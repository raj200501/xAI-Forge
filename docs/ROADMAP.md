# Roadmap

## v0.2 (Flight Recorder Upgrade)

- Flight Recorder UI redesign (dark mode, tabs, filters).
- Live run streaming from the UI.
- Trace comparison view.
- API metadata endpoints (`/api/tools`, `/api/providers`, `/api/schema/events`).
- Expanded documentation and community guidelines.

## v0.3 (Planned)

- Trace export (JSON, CSV, OpenTelemetry).
- Pluggable tool/provider discovery.
- Packaged desktop UI (Tauri/Electron).
- Trace diff annotations with event-level deltas.
- Signed trace bundles for provenance.
